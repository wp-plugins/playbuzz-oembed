Add Playbuzz to the WordPress oEmbed providers

== Description ==

The playbuzz plugin enables site owners to easily embed Playful Content items such as quizzes, lists and polls.

Upgrade your site to include highly engaging viral content, using playbuzz's Playful Content. You can have all this content on your site in minutes!

Embed content using nothing but a URL. Just copy the item/section URL from [playbuzz.com](https://www.playbuzz.com) and paste it to your post.

== Installation ==

= Minimum Requirements =
* WordPress version 3.5 or greater.
* PHP version 5.2.4 or greater.
* MySQL version 5.0 or greater.

= Recommended  Requirements =
* The latest WordPress version.
* PHP version 5.4 or greater.
* MySQL version 5.5 or greater.

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "Playbuzz oEmbed".
3. Install playbuzz plugin by pressing the plugin "Install" button.
4. Activate the playbuzz plugin.

= Updating =
* Use WordPress automatic updates to upgrade to the latest version. Ensure to backup your site just in case.

== Frequently Asked Questions ==

= What is Playbuzz? =

Playbuzz is an open network for end users, publishers, bloggers and brands to create and share playful content items such as quizzes, lists and polls.
